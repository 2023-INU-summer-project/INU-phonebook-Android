# INU-phonebook-Android

### __Commit Message Convention__
작성된 형식에 따라 Commit Message를 작성

#### type
- feat : 새로운 기능 추가
- fix : 오류 수정
- docs : 문서 수정
- refactor : 코드 리팩토링
- design : 사용자 UI 변경
- gradle : gradle 환경 수정
